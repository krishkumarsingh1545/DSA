## 🔑 Key Concepts

- **Prefix Sum:** Precompute cumulative sums to answer range queries in O(1)
- **Two-pointer on arrays:** Reduces O(n²) brute force to O(n) for many problems
- **In-place operations:** Modify the array without extra space (O(1) space)

## 🧠 Practice Tips

- Start with basic traversal, then move to prefix sums
- Practice in-place reversal, rotation, and rearrangement
- Always consider edge cases: empty array, single element, all same elements
