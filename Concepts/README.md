# 📖 Concepts

This directory contains **theory and concept implementations** for each data structure and algorithm topic. Every folder includes clean, well-commented code in C and Python.

## 📁 Folders

| Folder | Topic | Roadmap Phase |
|--------|-------|--------------|
| [Arrays/](./Arrays/) | 1D & 2D arrays, prefix sum, in-place operations | Phase 0 |
| [LinkedLists/](./LinkedLists/) | Singly, doubly, and circular linked lists | Phase 0 |
| [Stack/](./Stack/) | Array & linked list based stacks, monotonic stack | Phase 0 |
| [Queue/](./Queue/) | Circular queue, deque, queue using 2 stacks | Phase 0 |
| [Sorting/](./Sorting/) | Bubble, selection, insertion, merge sort, quick sort | Phase 1 |
| [Searching/](./Searching/) | Binary search, search on answer space | Phase 1 |

## 📝 Code Organization

Each folder may contain:
```
TopicName/
├── C/           # C implementations
├── Python/      # Python implementations
└── README.md    # Topic-specific notes
```

## 💡 Approach

- **Understand the concept** before coding — draw diagrams, trace through examples
- **Implement from scratch** in both C and Python without looking at notes
- **Time yourself** to build speed and confidence
