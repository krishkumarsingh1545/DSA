# 🔗 Linked Lists

> **Roadmap Phase:** 0 — Revision Sprint
> **Languages:** C · Python

---

## 🎯 Topics Covered

### Singly Linked List
- Node creation and traversal
- Insertion (beginning, end, at position)
- Deletion (by value, by position)
- Reversal (iterative & recursive)

### Doubly Linked List
- Forward and backward traversal
- Insertion and deletion operations
- Advantages over singly linked list

### Circular Linked List
- Circular singly and doubly linked lists
- Detecting loops (Floyd's cycle detection)
- Applications: round-robin scheduling

---

## 🔑 Key Concepts

- **Pointer manipulation (C):** `struct Node*` — master `next` and `prev` pointer operations
- **Class-based nodes (Python):** Clean OOP approach with `__init__`, `__repr__`
- **Runner technique:** Use fast/slow pointers for middle element, cycle detection
- **Dummy head node:** Simplifies edge cases in insertion/deletion

## 🧠 Practice Tips

- Code each operation from scratch — don't rely on built-in lists
- Draw diagrams for every pointer operation before writing code
- Always handle edge cases: empty list, single node, head/tail operations
