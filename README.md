# 🚀 DSA — Data Structures & Algorithms

A structured, concept-first journey through Data Structures and Algorithms in **C** and **Python**, following the **Striver A2Z DSA Sheet** roadmap.

---

## 📋 Roadmap Phases

| # | Phase | Topics | Timeline |
|---|-------|--------|----------|
| 0 | **Revision Sprint** | Arrays, Linked List, Stack, Queue | 1–2 weeks |
| 1 | **Core Foundations** | Recursion, Sorting, Hashing, Binary Search, Strings | 3–4 weeks |
| 2 | **Two-Pointer & Bit Tricks** | Sliding Window, Two Pointer, Bit Manipulation, Kadane's | 2–3 weeks |
| 3 | **Trees & Heaps** | Binary Tree, BST, Heaps, Greedy | 4–5 weeks |
| 4 | **Graphs** | BFS, DFS, Shortest Path, Disjoint Set, MST | 4–5 weeks |
| 5 | **Dynamic Programming** | 1D DP, 2D DP, DP on Strings & Trees | 5–6 weeks |
| 6 | **Advanced Structures** | Trie, Segment Tree, Advanced Stack/Queue, Interview Patterns | 3–4 weeks |

> **Estimated total:** 18–25 weeks (relaxed) · 12–16 weeks (intensive)

## 🏁 Getting Started

```bash
# Clone the repository
git clone https://github.com/<your-username>/DSA.git
cd DSA

# Compile & run a C file
gcc Concepts/Arrays/C/array1D.c -o array1D
./array1D

# Run a Python file
python Concepts/Arrays/Python/arrays.py
```
