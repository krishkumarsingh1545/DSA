# 🏋️ Practice

This directory contains **problem-solving code and practice exercises** organized by topic. Each folder holds solutions to common DSA problems in C and Python.

## 📁 Folders

| Folder | Topic | Focus |
|--------|-------|-------|
| [Arrays/](./Arrays/) | Array problems | Subarray sums, rearrangement, two-pointer |
| [LinkedLists/](./LinkedLists/) | Linked list problems | Reversal, merge, cycle detection |
| [Stack/](./Stack/) | Stack problems | Balanced brackets, next greater element |
| [Queue/](./Queue/) | Queue problems | BFS-based, sliding window max |
| [Sorting/](./Sorting/) | Sorting problems | Custom sorts, merge intervals |
| [Searching/](./Searching/) | Search problems | Binary search variants, answer space |
| [Patterns/](./Patterns/) | Pattern printing | Nested loops, star/number patterns |

## 📝 File Naming Convention

```
ProblemName.c      # C solution
ProblemName.py     # Python solution
```

## 💡 Approach

1. **Read the problem** — understand inputs, outputs, and constraints
2. **Brute force first** — get a working solution, then optimize
3. **Code in C first** — forces you to think about memory and pointers
4. **Rewrite in Python** — practice concise, Pythonic solutions
5. **Analyze complexity** — always note time and space complexity
