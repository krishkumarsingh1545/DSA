## 📌 Problem Checklist

### Basic Operations
- [ ] Reverse a singly linked list (iterative)
- [ ] Reverse a singly linked list (recursive)
- [ ] Find the middle of a linked list
- [ ] Detect a cycle in a linked list (Floyd's)
- [ ] Find the starting point of a cycle

### Intermediate
- [ ] Merge two sorted linked lists
- [ ] Remove N-th node from end
- [ ] Add two numbers represented as linked lists
- [ ] Palindrome linked list check
- [ ] Intersection point of two linked lists

### Doubly Linked List
- [ ] Insert at head, tail, and position
- [ ] Delete from head, tail, and position
- [ ] Reverse a doubly linked list

---

## 🔑 Patterns to Master

| Pattern | Application |
|---------|------------|
| **Fast & Slow pointer** | Middle element, cycle detection |
| **Dummy head** | Simplify insertion/deletion edge cases |
| **Recursive reversal** | Elegant but uses O(n) stack space |
| **Runner technique** | Two pointers at different speeds |

## 🧠 Tips

- Draw the pointer changes on paper before coding
- Edge cases: empty list, single node, two nodes
- In Java, let the GC handle memory — focus on null-safety (`NullPointerException` prevention)
