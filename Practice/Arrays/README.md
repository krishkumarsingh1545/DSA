# 📦 Array Practice Problems

> **Roadmap Phase:** 0 & 2
> **Languages:** Java · Python

---

## 📌 Problem Categories

### Phase 0 — Revision
- [ ] Reverse an array in-place
- [ ] Find the maximum and minimum element
- [ ] Rotate array by K positions
- [ ] Move all zeros to the end
- [ ] Prefix sum queries

### Phase 2 — Two Pointer & Sliding Window
- [ ] Two sum (sorted array)
- [ ] Remove duplicates from sorted array
- [ ] Maximum subarray sum (Kadane's algorithm)
- [ ] Subarray sum equals K (hashmap trick)
- [ ] Longest subarray with sum ≤ K

---

## 🔑 Patterns to Master

| Pattern | When to Use |
|---------|------------|
| **Two Pointer** | Sorted arrays, pair finding, merging |
| **Sliding Window** | Subarray/substring with constraints |
| **Prefix Sum** | Range sum queries, subarray sums |
| **Kadane's** | Maximum subarray problems |
| **Hashing** | Frequency count, subarray sum = K |

## 🧠 Tips

- Always ask: *"Is the array sorted?"* — this unlocks two-pointer and binary search
- Kadane's algorithm is a must-know — understand the intuition, not just the code
- Practice both Java (`int[]`, `ArrayList`) and Python (list slicing) approaches
